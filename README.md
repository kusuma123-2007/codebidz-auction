# CodeBidz Auction Platform

## 📌 Project Overview

CodeBidz Auction Platform is a web-based auction system that allows users to create auctions and place bids on items in real time. The platform manages bidding processes and determines the highest bidder when the auction ends.

This project demonstrates a complete auction workflow including product listing, bidding functionality, and bid tracking.

---

## 🚀 Features

* User-friendly auction platform
* Users can place bids on items
* Real-time bidding system
* Automatic highest bid tracking
* Auction closing mechanism
* Outbid notifications
* Simple and responsive user interface

---

## 💡 How the System Works

1. Users visit the auction platform.
2. Available items are displayed for bidding.
3. Users place bids on items.
4. The system records and compares bids.
5. The highest bid becomes the current winning bid.
6. When the auction ends, the highest bidder wins the item.
